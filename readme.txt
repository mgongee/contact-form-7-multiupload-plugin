== Description ==

### Adds Multiupload Fields to Contact Form 7

This plugin adds a new field to the Contact Form 7 plugin that enables the user to upload multiple files at once. The plugin uses the JavaScript library "DropzoneJS" (http://www.dropzonejs.com/).

This plugin requires version 3.9 or higher of the "Contact Form 7" plugin.

== Installation ==

This plugin requires the Contact Form 7 plugin.

1. Upload plugin files to your plugins folder, or install using WordPress' built-in Add New Plugin installer
1. Activate the plugin
1. Edit a form in Contact Form 7
1. Choose "multiupload" tag in the tag list
1. Fill field in the pop-up window and place tag in the form
